# Figures

This folder contains representative figures generated by the Python scripts included in this repository.

Representative outputs include:

- Radiative forcing time-series (1980–2024)
- Seasonal average radiative forcing over India (1980-2024)
- Sample geospatial radiative forcing maps over India (1980-2024)

# Note
- Sample geospatial radiative forcing maps over India (1980-2024)

The complete set of monthly maps (1980–2024) is not included because it consists of hundreds of images. All figures can be regenerated using the scripts in the `src` directory and the original MERRA-2 datasets.
